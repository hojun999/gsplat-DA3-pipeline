# Legacy Pipelines

This directory contains pipeline assets archived before the Stray Scanner and
DN-Splatter rewrite.

## Directories

- `pre_stray_dn_splatter_pipeline/`
  - The previously active Nerfstudio Splatfacto pipeline.
  - Includes the older DA3-to-gsplat experiments and generated notebooks.
- `historical/`
  - Older snapshots retained only for reference.

New Stray Scanner and DN-Splatter files should be created at the repository
root or in a new project directory. Do not add new implementation files to this
legacy directory.
